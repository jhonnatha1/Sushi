# Sushi

Site estático (HTML/CSS/JS + carrossel React).

## Deploy no Render

1. **Dashboard Render:** New → **Static Site** → conecta o repositório `jhonnatha1/Sushi`.
2. **Build Command:**  
   `cd sobre-carousel && npm install && npm run build`
3. **Publish Directory:**  
   `.` (ponto = raiz do repo, onde está o `index.html`).  
   **Não uses** `Sushi` – essa pasta não existe no clone; a raiz do repo já é o projeto.
4. Guarda e faz Deploy.

O `render.yaml` na raiz já define isto; se criares o serviço a partir do repositório, o Render pode usar este ficheiro.
