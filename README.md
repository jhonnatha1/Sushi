# Sushi

Site estático (HTML/CSS/JS + carrossel React).

## Deploy no Render

1. **Dashboard Render:** New → **Static Site** → conecta o repositório `jhonnatha1/Sushi`.
2. **Build Command:**  
   `npm install && npm run build`
3. **Publish Directory:**  
   `public`  
   (A pasta `public` é criada pelo build; não uses "Root" nem ".", usa exatamente `public`.)
4. Guarda e faz **Deploy**.

Se o Render pedir acesso ao repo, autoriza em GitHub → Settings → Applications.
